# Milestone v1.8.2: VMware UI Settings

**Status:** Complete
**Phases:** 14
**Total Plans:** 2

## Overview

Expose VMware network configuration (NetworkType, NicType) in the UI, following the same pattern as v1.8.1's IncludePreviewUpdates feature: UI controls, config persistence, and migration for existing configs.

## Phases

### Phase 14: VMware UI Settings

**Goal**: Expose VMware network configuration in UI with config migration

**Depends on**: Phase 13 (v1.8.1 complete)
**Requirements**: VMUI-01, VMUI-02, VMUI-03, VMUI-04, VMUI-05, VMUI-06
**Success Criteria** (what must be TRUE):
  1. User can select VMware network type from dropdown when VMware is selected
  2. User can select VMware NIC type from dropdown when VMware is selected
  3. Dropdowns are disabled when Hyper-V is selected
  4. Settings persist after save/load config cycle
  5. Existing configs without VMwareSettings get defaults on load
**Plans:** 2 plans
